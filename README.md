# IdeaForge AI

🚀 **AI-Powered Startup-Grade Project Planner for Students**

IdeaForge AI is a full-stack intelligent project planning platform that helps students transform project ideas into complete, portfolio-ready solutions. Powered by Google Gemini AI, it generates project scopes, roadmaps, architecture diagrams, dataset recommendations, and viva preparation materials.

## Features

* 🤖 AI-powered project generation
* 📊 Originality scanner to detect overused project ideas
* 🏗️ Interactive software architecture visualization
* 🎓 Viva preparation with categorized Q&A
* 📄 PDF project report generation
* 💾 Saved project library with search functionality

## Local Setup

### Backend

```bash
cd backend
python -m venv .venv
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
